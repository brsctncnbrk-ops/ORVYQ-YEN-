# Phase 0 — File Classification

Every file/directory inspected in the golden repo, classified as one of: **canonical source**,
**generated derived output**, **QA report**, **runtime cache**, **diagnostic artifact**,
**temporary proof-only file**, **obsolete file**. Source: direct inspection of
`brsctncnbrk-ops/YouTube_pepline` at `9affbd2494d8197a564c4a552b879fadb0e14a4a`.

Classification determines migration handling in later phases:
canonical source → recovered into the new repo's `templates/`/`schemas`/`scripts`/`projects`;
generated derived output → never hand-copied, always regenerated by the canonical pipeline;
QA report → never hand-copied, always regenerated by the QA chain;
runtime cache / diagnostic artifact / temporary proof-only file / obsolete file → **not
imported**.
